# vux-install
```$xslt
vux微信项目
```
#### 安装VUX
 ```$xslt
        cnpm install vux --save
```
#### 使用 vue-cli 工具和 airyland/vux2 模板快速初始化项目
```$xslt
        cnpm install vue-cli -g // 如果还没安装
        vue init airyland/vux2 projectPath
        
        cd projectPath
        cnpm install
        npm run dev
```
